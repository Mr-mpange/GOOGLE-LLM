COMMON ISSUES:
================================================================================

Issue: "gcloud is not recognized"
Fix: Install gcloud CLI (INSTALL_GCLOUD.txt)

Issue: "Unable to authenticate"
Fix: Run FIX_AUTH_NOW.bat or gcloud auth application-default login

Issue: "Billing not enabled" (403)
Fix: Enable billing (ENABLE_BILLING.txt), wait 2-5 minutes

Issue: "Port already in use"
Fix: Run STOP_ALL.bat
