FIRST TIME SETUP (REQUIRED):
================================================================================

Step 1: Install Node.js
  Download: https://nodejs.org

Step 2: Install gcloud CLI
  Download: https://cloud.google.com/sdk/docs/install
  See: INSTALL_GCLOUD.txt for detailed steps

Step 3: Authenticate
  Easy way: Run FIX_AUTH_NOW.bat (Windows)
  Manual:
    gcloud auth application-default login
    gcloud services enable aiplatform.googleapis.com --project=rosy-stronghold-475804-s7

Step 4: Enable Billing (Get $300 free credit!)
  Visit: https://console.developers.google.com/billing/enable?project=rosy-stronghold-475804-s7
  See: ENABLE_BILLING.txt for details
  
  Wait 2-5 minutes after enabling billing!

TROUBLESHOOTING:
================================================================================

"gcloud is not recognized"
‚Üí Install gcloud CLI (see INSTALL_GCLOUD.txt)

"Unable to authenticate"
‚Üí Run: gcloud auth application-default login

"Billing not enabled" (403 error)
‚Üí Enable billing (see ENABLE_BILLING.txt)
‚Üí Wait 2-5 minutes after enabling

"Port already in use"
‚Üí Run: STOP_ALL.bat

Backend starts but prompts fail
‚Üí Check: gcloud --version (installed?)
‚Üí Check: gcloud auth application-default print-access-token (authenticated?)
‚Üí Check: Billing enabled in Google Cloud Console?
‚Üí Wait 2-5 minutes after enabling billing
